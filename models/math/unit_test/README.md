# Unit Test - With Googletest
## Usage
1. Make sure ```sirius-simulation/googletest``` is available
    - If not, ```git clone git@github.com:google/googletest.git```
2. ```make``` in this directory
3. Run ./<testname>Test
4. Enjoy the results.
